# Clasificados shared

Cross-category concerns that are not yet migrated live primarily under:

- `app/clasificados/config/` — routing/category config, taxonomy re-exports
- `app/clasificados/lib/` — draft storage, listing view-model helpers
- `app/clasificados/components/` — grids, listing shell, publish UI shared with multiple categories

Category-owned modules should live under `en-venta/`, `rentas/`, or `bienes-raices/` as they are extracted.
