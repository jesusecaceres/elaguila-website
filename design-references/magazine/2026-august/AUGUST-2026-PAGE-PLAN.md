### AUGUST STARTER ISSUE: 12 PAGES

**Why 12 pages:**

- Enough to feel like a **real magazine** (cover, ads, editorial, close)
- **Manageable** for first structured August production cycle
- **Safe for print proofing** — fewer pages to QA before print shop
- **Safe for FlipHTML5 testing** — consistent page order without June-scale bulk
- **Easier to translation-proof** before scaling to 16 or 24 pages

## Translation-Safe Notes By Page Type

| Page Type | Main Translation Risk | Required Layout Rule |
|-----------|----------------------|----------------------|
| Cover | Headline length expansion | Short issue label; controlled hero text; extra margin on title block |
| House ad | CTA + body expansion | Wider text boxes; short CTA labels; Suite 201 footer with room |
| Food ad | Menu items, ingredients, CTA | Modular item lines; readable client contact; no tiny price text |
| Service ad | Pricing, service list | Bullet modules; horizontal room on CTA buttons |
| Event page | Dates, venues, URLs | Large row height in event list; verify dates before export |
| Recipe page | Ingredients + steps | HIGH risk — numbered steps with 15–25% extra vertical space |
| Wellness page | Long paragraphs | Break into short blocks; avoid narrow columns |
| Finance page | Tip lists | Bullet/card modules; no paragraph walls |
| Culture page | Names, titles, credits | Modular captions; avoid micro-type |
| Resources table page | Row density | HIGH risk — large rows, wide columns, no cramped tables |
| Back cover | CTA + contact | Short headline; QR in safe zone; Suite 201 contact bar spaced |

**Universal rules:**

- Fewer long paragraphs
- Wider text boxes
- Short CTA labels with horizontal room
- Readable contact info at normal view
- No tiny footer text
- Room for Portuguese/English expansion
- Resource tables need **large rows**

- All copy reviewed **manually** before Spanish final and print
- Final PNG/PDF export comes **after** Spanish QA and optional EN/PT proofs
- Fix **source layout** if translation breaks — do not patch translated exports as primary fix

4. **Generate / proof English pages** → `04-english-proof/`

7. **Export print PDF** → `06-print-ready-pdf/`
8. **Export FlipHTML5 package** → `07-fliphtml5-export/`
9. **Only then** decide website / `public/` placement (separate gate)
